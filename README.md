# Pixel Font Renderer

A toy tool for turning TTF fonts into pixel grids. 18 built-in blackletter & gothic fonts, with support for any Google Font.

Inspired by the amazing designer [Cj Cawley](https://www.youtube.com/watch?v=d3jPRG9K7aw) and his work on pixelized blackletter fonts.

**[Live Demo](https://experiment-pixelfont.mitchmalone.com/)**

## Running Locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Features

- Render any character as a pixel grid from 8×8 to 64×64
- 18 built-in blackletter, gothic, and ornate fonts
- Add any Google Font by name
- Adjustable scale, threshold, offset, gap, and color
- Export as SVG, PNG, or CSS box-shadow
- Side-by-side comparison with smooth vector rendering
- Resolution comparison across multiple grid sizes
